task structure:
status Enabled The specified task is enabled.
Disabled The specified task is disabled.
actionType Move Move (archive) the specified folders.
Delete Permanently delete the specified folders.
source E:\\temp Example: Source folder
destination E:\\temp\\archive Example: Destination folder
action/id TotalUsedSpace Action will execute if total used space on specifed DISK is over the value specifed in action/value
action/id TotalFolderUsedSpace Action will execute if total used space in specifed source folder is over the value specifed in action/value
TotalFreeSpace Action will execute if total free space on specified DISK is under the value specified in action/value
TotalFolderCount Action will execute if total foldercount in specifed source folder is over the value specifed in action/value
DateOlder Action will execute if one or more folders specified in source folder is older than the value specifed in action/value
action/value Number Example: Number or number of days if action is DateOlder
action/minFolderAction Number Action will execute only if the number of folders is greather than specified.
